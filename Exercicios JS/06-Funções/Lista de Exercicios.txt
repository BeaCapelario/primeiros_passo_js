Jogo de Adivinhação Inteligente: ​
Crie uma função chamada jogar_adivinhacao que implementa um jogo de adivinhação. O computador deve escolher um número aleatório entre 1 e 100. O jogador tem 10 tentativas para adivinhar o número. A cada tentativa, o programa deve informar se o número digitado é maior ou menor que o número secreto. Se o jogador acertar, o programa deve exibir uma mensagem de parabéns e informar o número de tentativas que ele utilizou. ​
Desafio: A função deve implementar uma "dica inteligente" para o jogador. Se o jogador errar, a função deve sugerir um intervalo menor para o próximo palpite (por exemplo, "O número está entre 50 e 70"). ​
